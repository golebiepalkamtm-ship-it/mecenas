# Asesor AI - Wersje Mobilne

## 📱 Android APK
**Plik**: `AsesorAI_Android.apk`
**Rozmiar**: ~250MB (zawiera wszystkie modele AI i PDF-y)
**Wymagania**: Android 8.0+

### Instalacja:
1. Pobierz APK na urządzenie Android
2. W ustawieniach włącz "Instalację z nieznanych źródeł"
3. Kliknij plik APK i zainstaluj
4. Uruchom aplikację

## 🍎 iOS IPA
**Plik**: `AsesorAI_iOS.ipa`
**Rozmiar**: ~280MB (zawiera wszystkie modele AI i PDF-y)
**Wymagania**: iOS 14.0+

### Instalacja (wymaga macOS + Xcode):
1. Podłącz urządzenie iOS do Mac'a
2. Otwórz Xcode
3. Przeciągnij plik IPA do biblioteki urządzeń
4. Wybierz urządzenie i zainstaluj

## ✨ Funkcje wszystkich wersji:
- 🤖 AI Ekspert Prawny oparty na 3 źródłach prawnych
- 📚 Wszystkie PDF-y wbudowane (offline)
- 🎨 Nowoczesny interfejs z ciemnym motywem
- ⚡ Szybkie odpowiedzi dzięki lokalnym modelom
- 🔒 Prywatność - wszystko działa lokalnie

## 🖥️ Wersja Desktop:
**Plik**: `AsesorAI.exe`
**Rozmiar**: ~200MB
**Wymagania**: Windows 10+

Po prostu kliknij dwukrotnie `uruchom.bat` lub `AsesorAI.exe`

## 🌐 Wersja Web:
Uruchom lokalnie: `streamlit run app.py`
